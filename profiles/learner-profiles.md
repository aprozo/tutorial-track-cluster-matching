---
title: 'Learner Profiles'
---

This lesson targets members of the ePIC collaboration — students, postdocs, and researchers — who
have a working `eic-shell` environment and want to learn how to use the PODIO interface to analyze
the reconstructed simulation output produced by EICrecon.  Learners are expected to have some
familiarity with CERN ROOT and/or Python and with the basics of the reconstruction output tree
(covered in the [analysis tutorial](https://eic.github.io/tutorial-analysis/)).  No prior
experience with PODIO or the EIC data model (EDM4eic/EDM4hep) is assumed.
