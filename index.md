Welcome to the EIC/ePIC tutorial on PODIO and track-cluster matching! This tutorial will
illustrate the basics of PODIO and how to use it in an analysis setting via track-cluster 
matching.

> ## Prerequisites
> ----------------
> You will need experience/knowledge with the following topics:
>   - Using CERN ROOT and/or Python;
>   - Working on the command line in UNIX environments;
>   - And the eic software environment.
>
> This builds on other tutorials in the EIC series:
>   - [Setting up your environment](https://eic.github.io/tutorial-setting-up-environment/)
>   - [Accessing EIC files](https://eic.github.io/tutorial-file-access/)
>   - [Analysis basics](https://eic.github.io/tutorial-analysis/)
>
> Further information is included in other tutorials:

>   - [Kinematic Reconstruction](https://eic.github.io/tutorial-kinematic-reconstruction/)

>   - [Geometry Development with DD4hep](https://eic.github.io/tutorial-geometry-development-using-dd4hep/)
>   - [Modifying Geometry, Digitization, Etc](https://eic.github.io/tutorial-modifying-geometry-digitization-etc/)
>   - [Developing Benchmarks](https://eic.github.io/tutorial-developing-benchmarks/)
